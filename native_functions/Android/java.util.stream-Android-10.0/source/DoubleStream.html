/**@class java.util.stream.DoubleStream
 implements java.util.stream.BaseStream

 A sequence of primitive double-valued elements supporting sequential and parallel
 aggregate operations.  This is the {@code double} primitive specialization of
 {@link java.util.stream.Stream}.

 <p>The following example illustrates an aggregate operation using
 {@link java.util.stream.Stream} and {@link java.util.stream.DoubleStream}, computing the sum of the weights of the
 red widgets:

 <pre>{@code
     double sum = widgets.stream()
                         .filter(w -> w.getColor() == RED)
                         .mapToDouble(w -> w.getWeight())
                         .sum();
 }</pre>

 See the class documentation for {@link java.util.stream.Stream} and the package documentation
 for <a href="package-summary.html">java.util.stream</a> for additional
 specification of streams, stream operations, stream pipelines, and
 parallelism.

 @since 1.8

 <pre>{@code
