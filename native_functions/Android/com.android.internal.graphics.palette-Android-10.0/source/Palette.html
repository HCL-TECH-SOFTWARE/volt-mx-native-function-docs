 generated Palette. See that class' documentation for more information.
 <p>
 Generation should always be completed on a background thread, ideally the one in
 which you load your image on. {@link com.android.internal.graphics.palette.Palette.Builder} supports both synchronous and asynchronous
 generation:

 <pre>
 // Synchronous
