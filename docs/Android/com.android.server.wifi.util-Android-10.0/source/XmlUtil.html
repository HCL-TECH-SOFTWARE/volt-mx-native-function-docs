 This utility is designed for formatting the XML into the following format:
 <Document Header>
  <Section 1 Header>
   <Value 1>
   <Value 2>
   ...
   <Sub Section 1 Header>
    <Value 1>
    <Value 2>
    ...
   </Sub Section 1 Header>
  </Section 1 Header>
 </Document Header>
