 UsbRequests can be used to transfer data on bulk and interrupt endpoints.
 Requests on bulk endpoints can be sent synchronously via {@link android.hardware.usb.UsbDeviceConnection#bulkTransfer}
 or asynchronously via {@link #queue} and {@link android.hardware.usb.UsbDeviceConnection#requestWait}.
 Requests on interrupt endpoints are only send and received asynchronously.
